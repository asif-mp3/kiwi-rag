# Credentials Directory

This directory contains Google Sheets API credentials.

## Setup Instructions

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select an existing one
3. Enable the Google Sheets API
4. Create credentials (Service Account)
5. Download the JSON key file
6. Place the JSON file in this directory
7. Update your `.env` file with the path to the credentials file

**Note**: The actual credential files are excluded from version control for security reasons.
